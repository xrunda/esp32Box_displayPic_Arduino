# MCP Client 组件迁移 - 快速开始

## 使用方法

**只需使用一个提示词即可！** 推荐使用完整版。

## 推荐方式：完整版提示词（一次性提交）

直接复制 `AI_MIGRATION_PROMPT.md` 中的完整版提示词，一次性提交给 AI。这样 AI 可以获得所有信息，一次性完成迁移。

## 何时使用简化版

只有在以下情况才使用简化版：
- AI 已经熟悉 ESP-IDF 组件迁移流程
- 你只需要快速复制文件，不需要详细指导
- 你会在后续对话中补充细节

## 提示词选择建议

### 场景 1：新项目，第一次迁移
→ **使用完整版提示词**

### 场景 2：有经验的开发者，只需要快速迁移
→ **使用简化版提示词**

### 场景 3：不确定该用哪个
→ **使用完整版提示词**（更保险）

## 完整版提示词位置

文件：`AI_MIGRATION_PROMPT.md`  
从 "## 完整提示词（可直接复制使用）" 开始复制到文件末尾。

## 简化版提示词位置

文件：`AI_MIGRATION_PROMPT.md`  
在文件末尾的 "## 简化版提示词（快速迁移）" 部分。
